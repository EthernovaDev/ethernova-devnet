devnet: reduce State Lifecycle tier thresholds for faster testing

Reduce Active/Warm/Cold tier thresholds to significantly speed up Phase 5 testing on devnet.

Changes:
- ActiveTierBlocks: 100 -> 10
- WarmTierBlocks:   500 -> 25
- ColdTierBlocks:   1000 -> 50

Rationale:
Current devnet values require ~3 hours to reach Archived state (1000 blocks at ~10-11s/block),
making iterative testing (tier transitions, witness restore, stress tests) extremely slow.

With the new values:
- Full lifecycle (Active -> Archived) completes in ~9 minutes
- Enables rapid iteration and debugging for Phase 5 features

IMPORTANT:
These values are NOT intended for mainnet.
Mainnet should use significantly higher thresholds (e.g. 100k / 1M / 10M blocks)
to preserve long-term state economics and realistic lifecycle behavior.

Reminder:
Revert or adjust these parameters before mainnet deployment.
